#### 🚀 Como executar o projeto

##### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
- **[Git](https://git-scm.com)**,
[JAVA 11]() , 
[Node.js](https://nodejs.org/en/).
Além disto é bom ter um editor para trabalhar com o código como [VSCode](https://code.visualstudio.com/) e o [IntelliJ]()
---

#### 👯 Clonando o repositório

```bash
# Clone este repositório
$ git clone https://github.com//PhilipeGama/nomeprojeto

# Acesse a pasta do projeto no terminal/cmd
$ cd nomeprojeto
```

#### 📁 Rodando a Aplicação

##### Frontend
```bash
# Instale as dependências
$ yarn or npm install

# Roda projeto Angular
$ ng serve
```

##### Backend
```bash
# Rodar projeto Spring Boot
$ mv spring-boot:run

```

---

#### 🔎 Comandos

- `start`: Executa a aplicação em `localhost:3000`
- `build`: Cria a Build do projeto
